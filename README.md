MAE 472 (Senior Design Capstone Project for CSULB) Automatic Tracking Pyrometer program for a company called California Nanotechnologies (CalNano for short). 
ArUco_Dual_Tracker.py is a python program made to be ran on a Raspberry Pi 5 with 8 GB of RAM.
Arduino_Manual_Control.ino is for manual takeover and initial adjustments through an Arduino UNO R3.
